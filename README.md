# 📚 题库学习助手

智能题库学习工具，支持多题库管理、错题集、弱项加强。

## ✨ 功能特点

- 📚 多题库管理（侧边栏切换）
- 📖 背诵模式 / ✍️ 答题模式
- ⭐ 标记收藏
- ❌ 错题集自动收录
- 💪 弱项加强（智能加权）
- 🎲 随机 / 🔄 还原顺序
- 🔍 关键词搜索
- 📊 学习统计（正确率、进度）
- 💾 本地存储（数据不丢失）
- 📱 手机端优化布局

## 🚀 在线体验

访问：[https://你的用户名.github.io/仓库名/](https://你的用户名.github.io/仓库名/)

## 📱 安装到手机

### Android
1. 用 Chrome 浏览器打开网址
2. 点击菜单 → "添加到主屏幕"

### iOS
1. 用 Safari 浏览器打开网址
2. 点击分享 → "添加到主屏幕"

## 📖 使用说明

1. 点击"📁 导入"按钮
2. 选择 Word 或 TXT 格式的题库文件
3. 开始学习！

## 🔧 题库格式

支持以下格式：

```
1. 这是题目内容？
A. 选项A
B. 选项B
C. 选项C
D. 选项D
答案：A

2. 填空题内容____和____。
答案：全面的，系统的

3. 判断题内容。
答案：正确
```
